# 🎨 Colour Palette Generator

A simple yet powerful web app that generates stunning colour palettes for designers and developers. Built with **HTML, CSS, and JavaScript**, it allows users to create random palettes, copy hex codes, and explore endless colour combinations.

---

## ✨ Features
- Generate random colour palettes instantly
- Copy hex codes with one click
- Responsive design for desktop and mobile
- Clean UI with modern styling
- Lightweight and fast — no external libraries

---

## 🛠️ Technologies Used
- **HTML5** for structure
- **CSS3** for styling and layout
- **JavaScript (ES6)** for functionality

---


## 📋 How to Use
1. Click **Generate** to create a new palette.
2. Hover over a colour to see its hex code.
3. Click the **Copy** button to copy the code to your clipboard.
4. Use the palette in your design projects.
